# 10-inch Mini-rack Mounts

A library of rack mounts for 10-inch mini server racks

Originally developed for my personal homelab. Every model includes preconfigured _.3mf_ print profiles for Bambu Lab
printers, alongside source _.f3d_ files and universal _.step_ files for full modification freedom
